# FoodPulse 🍽️

FoodPulse is a smart hostel mess management system designed to reduce food wastage in hostels and PG accommodations.

## Features

* Menu voting system for students
* Food feedback collection
* Food waste tracking
* Mess bill calculation
* Data-driven meal planning

## Tech Stack

* MySQL
* Java (Spring Boot planned)
* REST APIs

## Database Setup

Run the script:

```
database/database-setup.sql
```

to initialize the database.
